# **✈️ Cayman Airways Network Optimization**

Developed a data-driven optimization model to improve Cayman Airways’ international network while balancing operational efficiency and public service obligations. Used Python to analyze demand, resource utilization, and network performance, and automated data workflows with Make.

Applied Mixed-Integer Linear Programming (MILP) and Traveling Salesman Problem (TSP) models to optimize routing and fleet utilization for the Boeing 737-8 MAX, resulting in a more efficient and profitable network configuration.
